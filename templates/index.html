P (Proportional) — ตอบสนองทันทีตาม error · สูงเกิน → oscillate · ต่ำเกิน → sluggish
Pitch P ควรสูงกว่า Roll P ประมาณ 5–8%
I (Integral) — ต้าน drift + ลม · BF4.4+: 80–100 ปกติ
สูงเกิน → bounce หลัง flip · ต่ำเกิน → drift ใน wind
D (Derivative) — ลด overshoot + propwash · 5": 28–42 ปกติ
สูงเกิน → motor ร้อน · ต่ำเกิน → propwash · ปรับ D ก่อน P เสมอ

Gyro LPF1 · Nano/Whoop:250–300 · 5" Freestyle:200–250 · LR:120–150 Hz
เปิด RPM filter → ลด Gyro LPF1 ได้ → latency น้อยลง, motor เย็นลง
DTerm LPF1 · ตั้ง ~50-60% ของ Gyro LPF1 · 5":≈110 Hz · LR:≈90 Hz
Motor ร้อนหลังบิน → ลด DTerm LPF1 ลง 10–20 Hz ก่อน
Dynamic Notch · 1 notch=LR นิ่ง · 2–3 notch=freestyle/racing สั่น
ดู Blackbox spectrogram → ปรับ min/max Hz ตาม peak จริง
RPM Filter · ต้องการ BLHeli32 หรือ AM32 + Bidirectional DSHOT
ผล: noise ลดมาก · motor เย็น · สามารถเพิ่ม P/D ได้อีก
Anti-Gravity · 3–5=LR/freestyle · 5–7=freestyle throttle pump บ่อย
Nose dip/rise ตอน throttle chop → เพิ่มค่า Anti-Gravity
rpm_filter_min_hz · สูตร: RPM × 7 poles ÷ 60 × 0.22 (ที่ ~22% throttle)
ค่าต่ำเกิน = กรอง low freq ที่ไม่ใช่ motor noise

TUNING ROADMAP — ลำดับที่ถูกต้อง
STEP 01 — PRE-FLIGHT
Flash firmware → Paste CLI → ตรวจ motor direction → arm test → Motor test ใน Configurator
STEP 02 — BASELINE FLIGHT
บิน hover 1 pack · สังเกต hover throttle จริง ≈ {{ _hov|int }}% (คำนวณ: {{ _hov|int }}%) · ตรวจ motor temp (ควร <60°C)
STEP 03 — D TERM ก่อน (motor ร้อน + propwash)
Motor ร้อน → ลด D ทีละ 3 · Propwash หลัง flip → เพิ่ม D ทีละ 3
set d_roll  = {{ analysis.pid.roll.d }}  ← เริ่มที่นี่
set d_pitch = {{ analysis.pid.pitch.d }}
STEP 04 — P TERM (ตอบสนอง)
Motor หึ่งตอน throttle ขึ้น-ลง → ลด P ทีละ 3 · Pitch P > Roll P เสมอ
set p_roll  = {{ analysis.pid.roll.p }}
set p_pitch = {{ analysis.pid.pitch.p }}
STEP 05 — FILTER (latency + motor temp)
Motor เย็นดี → เพิ่ม Gyro LPF1 ขึ้น 10 Hz ทีละครั้ง
set gyro_lpf1_static_hz = {{ analysis.filter.gyro_lpf1 or 0 }}
set rpm_filter_min_hz   = {{ _rfmin }}
STEP 06 — FINE TUNE (TPA + FF + Rates)
TPA ลด P ที่ throttle สูง · FF ลด latency · Rates ปรับ stick feel
set tpa_rate          = {{ '55' if _sy=='racing' else '65' }}
set tpa_breakpoint    = 1300
set feedforward_roll  = {{ '120' if _sy=='racing' else '100' }}
set feedforward_pitch = {{ '120' if _sy=='racing' else '100' }}
